# Installation
## Reader Pi
- git clone https://github.com/gertlind/filamentDB-inky-display.git
- cd filamentDB-inky-display
- sudo apt update
- sudo apt full-upgrade -y
- sudo reboot
```bash
sudo apt install -y\
    git \
    pcscd \
    pcsc-tools \
    libpcsclite-dev \
    build-essential \
    python3-dev \
    python3-venv
```
